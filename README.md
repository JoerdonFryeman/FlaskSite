# FlaskSite template

## Example of a simple Flask website

Run the following commands to bootstrap your environment

```console
git clone https://github.com/JoerdonFryeman/FlaskSite
cd FlaskSite

python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

cp .env.template .env
```

Run the app locally:
source venv/bin/activate
python3 views.py
